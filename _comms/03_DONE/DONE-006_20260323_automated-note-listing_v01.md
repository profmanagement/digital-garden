## What was done

Updated the GitHub Actions workflow (`deploy.yml`) to automatically generate the note listing on the index pages (`en/index.html` and `nl/index.html`) using the new HTML structure and styling requested by the user. This ensures the note list is always up-to-date and formatted consistently without manual intervention.

- `.github/workflows/deploy.yml` — updated `build_notes_listing` function to generate the new HTML structure for note summaries, including the `{type}` badge.

- `content/nl/nl/index.md` — removed manual list to prevent duplication.

## Decisions made

**Automated Listing:** The list is generated by `deploy.yml` and injected into the index pages, replacing the manual list. This approach is more robust and less error-prone than manual updates.

**Type Badge:** Added the `{type}` badge (e.g., "SEEDLING") before the author line, as requested, to visually categorize the note maturity.
